# slab-build-utils

Build-time utilities for the Slab workspace.

## Role

`slab-build-utils` provides shared helpers used in `build.rs` scripts across the workspace, including native library linking helpers, path resolution for vendored artifacts, and compile-time feature detection.

## Type

Rust library crate (build support).
