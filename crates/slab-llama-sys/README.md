# slab-llama-sys

Unsafe FFI bindings to the llama.cpp native library.

## Role

`slab-llama-sys` is the `-sys` crate that links against the prebuilt llama.cpp shared libraries (vendored in `testdata/`) and exposes raw C bindings generated by `bindgen`. It is consumed exclusively by `crates/slab-llama`, which provides the safe Rust wrapper.

## Type

Rust library crate (native bindings / FFI).
