# slab-libfetch

Model artifact download library for Slab.

## Role

`slab-libfetch` handles fetching model weights and other large binary artifacts from remote sources. It provides resumable downloads, integrity verification, and progress reporting, and is used by `crates/slab-app-core` to support the model download workflow.

## Type

Rust library crate.
