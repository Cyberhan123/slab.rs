# slab-subtitle

Subtitle parsing library for Slab.

## Role

`slab-subtitle` provides subtitle file parsing and conversion utilities used in Slab's audio and video processing pipeline. It originated as a fork of [subparse](https://github.com/kaegi/subparse) with modifications to integrate with the Slab workspace.

Supported formats include SRT and other common subtitle container formats.
