# slab-whisper-sys

Unsafe FFI bindings to the whisper.cpp native library.

## Role

`slab-whisper-sys` is the `-sys` crate that links against the prebuilt whisper.cpp shared libraries and exposes raw C bindings generated by `bindgen`. It is consumed exclusively by `crates/slab-whisper`, which provides the safe Rust wrapper.

## Type

Rust library crate (native bindings / FFI).
