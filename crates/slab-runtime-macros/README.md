# slab-runtime-macros

Procedural macros for the Slab runtime core.

## Role

`slab-runtime-macros` provides derive macros and attribute macros used internally by `crates/slab-runtime-core` to reduce boilerplate in runtime handler and scheduler implementations.

## Type

Rust proc-macro crate.
